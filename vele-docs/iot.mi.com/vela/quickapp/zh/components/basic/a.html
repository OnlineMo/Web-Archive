# a

# 概述

超链接（默认不带下划线）

# 子组件

仅支持<span>

# 属性

支持通用属性

名称 类型 默认值 必填 描述
href string - 否 支持的格式参见页面路由中的 uri 参数。
额外的：
href 还可以通过“?param1=value1”的方式添加参数，参数可以在页面中通过this.param1的方式使用。使用this.param1变量时，需要在目标页面中在 public（应用外传参）或 protected（应用内传参）下定义 key 名相同的属性
示例：
<a href="/about?param1=value1">关于</a>

# 样式

支持text样式

支持通用样式

# 事件

支持通用事件

# 示例代码

<template>
  <div>
    <a class="link" href="/home">goHome</a>
    <a href="/home">
      <span class="link">使用span子组件</span>
    </a>
  </div>
</template>
