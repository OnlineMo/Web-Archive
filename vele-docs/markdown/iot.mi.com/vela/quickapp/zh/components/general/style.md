# [\#](style.html#通用样式){.header-anchor} 通用样式

通用样式，即所有组件都可以支持的样式。

它们均与 css
的属性样式用法保持一致，开发者可写在`内联样式`或`<style>`标签里，实现组件样式的定制化。

关于组件样式的设置，可以参考此[文档](../../guide/framework/style/page-style-and-layout.html)。

## [\#](style.html#示例代码){.header-anchor} 示例代码

::: {.language-html .extra-class}
``` language-html
<template>
  <div class="page">
    <div class="box-normal" style="background-color:#f76160"></div>
    <div class="box-normal"></div>
  </div>
</template>

<style>
  .page {
    padding: 30px;
    background-color: white;
  }

  .box-normal {
    background-color: #09ba07;
    width: 100px;
    height: 100px;
    border-radius: 8px;
    margin-right: 10px;
  }
</style>
```
:::
