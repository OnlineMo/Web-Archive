# [\#](device.html#设备信息-device){.header-anchor} 设备信息 device

## [\#](device.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.device" }
```
:::

## [\#](device.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import device from '@system.device' 
// 或 
const device = require('@system.device')
```
:::

## [\#](device.html#接口定义){.header-anchor} 接口定义

### [\#](device.html#device-getinfo-object){.header-anchor} device.getInfo(OBJECT) {#device-getinfo-object}

获取设备信息

#### [\#](device.html#参数){.header-anchor} 参数：

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](device.html#success-返回值){.header-anchor} success 返回值：

  参数值                                                                                                                             类型     说明
  ---------------------------------------------------------------------------------------------------------------------------------- -------- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  brand                                                                                                                              string   设备品牌
  manufacturer                                                                                                                       string   设备生产商
  model                                                                                                                              string   设备型号
  product                                                                                                                            string   设备代号
  osType                                                                                                                             string   操作系统名称
  osVersionName                                                                                                                      string   操作系统版本名称
  osVersionCode                                                                                                                      number   操作系统版本号
  platformVersionName                                                                                                                string   运行平台版本名称
  platformVersionCode                                                                                                                number   运行平台版本号
  language                                                                                                                           string   系统语言
  region                                                                                                                             string   系统地区
  APILevel[[2+]{.badge .apilevel style="vertical-align:top;" v-15b7b770=""}](../../guide/version/APILevel2.html){.apilevel-a}        number   框架api版本
  screenWidth                                                                                                                        number   屏幕宽
  screenHeight                                                                                                                       number   屏幕高
  screenDensity[[3+]{.badge .apilevel style="vertical-align:top;" v-15b7b770=""}](../../guide/version/APILevel3.html){.apilevel-a}   number   屏幕密度，即：设备像素比（device pixel ratio），是设备物理像素和逻辑像素（DP）的比值，其计算公式为：DPR = 设备 PPI / 160，PPI（pixels per inch）表示每英寸的像素数
  screenShape                                                                                                                        string   屏幕形状，可取值：rect 表示方形屏，circle 表示圆形屏，pill-shaped[[3+]{.badge .apilevel style="vertical-align:top;" v-15b7b770=""}](../../guide/version/APILevel3.html){.apilevel-a} 表示胶囊形屏
  deviceType[[2+]{.badge .apilevel style="vertical-align:top;" v-15b7b770=""}](../../guide/version/APILevel2.html){.apilevel-a}      string   设备类型，可取值：watch、band、smartspeaker

#### [\#](device.html#示例){.header-anchor} 示例：

::: {.language-javascript .extra-class}
``` language-javascript
device.getInfo({
  success: function(ret) {
    console.log(`handling success， brand = ${ret.brand}`)
  }
})
```
:::

### [\#](device.html#device-getdeviceid-object){.header-anchor} device.getDeviceId(OBJECT) {#device-getdeviceid-object}

获取设备唯一标识

#### [\#](device.html#权限要求){.header-anchor} 权限要求

获取设备信息

开发者需要在 manifest.json 里面配置权限：

::: {.language-json .extra-class}
``` language-json
{
  "permissions": [
    { "name": "hapjs.permission.DEVICE_INFO" }
  ]
}
```
:::

#### [\#](device.html#参数-2){.header-anchor} 参数： {#参数-2}

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](device.html#success-返回值-2){.header-anchor} success 返回值： {#success-返回值-2}

  参数值     类型     说明
  ---------- -------- --------------
  deviceId   String   设备唯一标识

#### [\#](device.html#示例-2){.header-anchor} 示例： {#示例-2}

::: {.language-javascript .extra-class}
``` language-javascript
device.getDeviceId({
  success: function (data) {
    console.log(`handling success: ${data.deviceId}`)
  },
  fail: function (data, code) {
    console.log(`handling fail, code = ${code}`)
  },
})
```
:::

获取存储空间的总大小

#### [\#](device.html#参数-4){.header-anchor} 参数： {#参数-4}

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
