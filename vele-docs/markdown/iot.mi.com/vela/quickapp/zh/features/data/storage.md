# [\#](storage.html#数据存储-storage){.header-anchor} 数据存储 storage

## [\#](storage.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.storage" }
```
:::

## [\#](storage.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import storage from '@system.storage' 
// 或 
const storage = require('@system.storage')
```
:::

## [\#](storage.html#方法){.header-anchor} 方法

### [\#](storage.html#storage-get-object){.header-anchor} storage.get(OBJECT) {#storage-get-object}

读取存储内容

#### [\#](storage.html#参数){.header-anchor} 参数：

  参数名     类型       必填   说明
  ---------- ---------- ------ -----------------------------------------------------------------------------
  key        String     是     索引
  default    String     否     如果 key 不存在，返回 default。如果 default 未指定，返回长度为 0 的空字符串
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](storage.html#success-返回值){.header-anchor} success 返回值：

key 对应的存储内容

#### [\#](storage.html#示例){.header-anchor} 示例：

::: {.language-javascript .extra-class}
``` language-javascript
storage.get({
  key: 'A1',
  success: function(data) {
    console.log('handling success')
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
:::

### [\#](storage.html#storage-set-object){.header-anchor} storage.set(OBJECT) {#storage-set-object}

修改存储内容

#### [\#](storage.html#参数-2){.header-anchor} 参数： {#参数-2}

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------------------------------------------------------
  key        String     是     索引
  value      String     否     新值。如果新值是长度为 0 的空字符串，会删除以 key 为索引的数据项
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](storage.html#示例-2){.header-anchor} 示例： {#示例-2}

::: {.language-javascript .extra-class}
``` language-javascript
storage.set({
  key: 'A1',
  value: 'V1',
  success: function(data) {
    console.log('handling success')
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
:::

### [\#](storage.html#storage-clear-object){.header-anchor} storage.clear(OBJECT) {#storage-clear-object}

清空存储内容

#### [\#](storage.html#参数-3){.header-anchor} 参数： {#参数-3}

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](storage.html#示例-3){.header-anchor} 示例： {#示例-3}

::: {.language-javascript .extra-class}
``` language-javascript
storage.clear({
  success: function(data) {
    console.log('handling success')
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
:::

### [\#](storage.html#storage-delete-object){.header-anchor} storage.delete(OBJECT) {#storage-delete-object}

删除存储内容

#### [\#](storage.html#参数-4){.header-anchor} 参数： {#参数-4}

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
  key        String     是     索引
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](storage.html#示例-4){.header-anchor} 示例： {#示例-4}

::: {.language-javascript .extra-class}
``` language-javascript
storage.delete({
  key: 'A1',
  success: function(data) {
    console.log('handling success')
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
