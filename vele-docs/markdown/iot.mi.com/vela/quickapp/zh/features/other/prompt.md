# [\#](prompt.html#弹窗-prompt){.header-anchor} 弹窗 prompt

## [\#](prompt.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.prompt" }
```
:::

## [\#](prompt.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import prompt from '@system.prompt' 
// 或 
const prompt = require('@system.prompt')
```
:::

## [\#](prompt.html#接口定义){.header-anchor} 接口定义

### [\#](prompt.html#prompt-showtoast-object){.header-anchor} prompt.showToast(OBJECT) {#prompt-showtoast-object}

显示 Toast 提示信息

#### [\#](prompt.html#参数){.header-anchor} 参数

  参数名     类型     必填   说明
  ---------- -------- ------ --------------------------------------------------------
  message    String   是     显示的文本信息
  duration   Number   否     显示持续时间，单位ms，默认值1500，建议区间：1500-10000

#### [\#](prompt.html#示例){.header-anchor} 示例：

::: {.language-javascript .extra-class}
``` language-javascript
prompt.showToast({
  message: 'Message Info',
  duration: 2000
})
```
