# [\#](event.html#事件-event){.header-anchor} 事件 event[[4+]{.badge .apilevel style="vertical-align:top;" v-15b7b770=""}](../../guide/version/APILevel4.html){.apilevel-a} {#事件-event}

## [\#](event.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.event" }
```
:::

## [\#](event.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import event from '@system.event' 
// 或 
const event = require('@system.event')
```
:::

## [\#](event.html#接口定义){.header-anchor} 接口定义

### [\#](event.html#event-publish-object){.header-anchor} event.publish (OBJECT) {#event-publish-object}

发布公共事件

#### [\#](event.html#参数){.header-anchor} 参数：

  参数名      类型     必填   说明
  ----------- -------- ------ ------------------------------------------------
  eventName   String   是     事件名称，公共事件保留名称被系统占用，请勿使用
  options     Object   否     事件参数

#### [\#](event.html#options-参数){.header-anchor} options 参数：

  参数名        类型     必填   说明
  ------------- -------- ------ ----------------------------------------------
  params        Object   否     事件参数
  permissions   Array    否     订阅者的权限，拥有权限的包才能收到发送的事件

#### [\#](event.html#系统支持的公共事件){.header-anchor} 系统支持的公共事件：

  系统内部事件名称              订阅者所需权限   说明
  ----------------------------- ---------------- --------------------------------------
  usual.event.BATTERY_CHANGED   无               电量改变，参数：level:0.0 - 1.0 之间
  usual.event.DISCHARGING       无               停止充电
  usual.event.CHARGING          无               开始充电

#### [\#](event.html#返回值){.header-anchor} 返回值：

无

#### [\#](event.html#示例){.header-anchor} 示例：

::: {.language-javascript .extra-class}
``` language-javascript
event.publish({
  eventName: 'myEventName',
  options: {
    params: { age: 10, name: 'peter' },
    permissions: ['com.example.demo']
  }
})
```
:::

### [\#](event.html#event-subscribe-object){.header-anchor} event.subscribe(OBJECT) {#event-subscribe-object}

订阅公共事件

#### [\#](event.html#参数-2){.header-anchor} 参数： {#参数-2}

  参数名      类型       必填   说明
  ----------- ---------- ------ ----------
  eventName   String     是     事件名称
  callback    Function   是     回调函数

#### [\#](event.html#回调参数){.header-anchor} 回调参数：

  参数名    类型     必填   说明
  --------- -------- ------ ----------------
  params    Object   否     事件参数
  package   String   否     事件推送者包名

#### [\#](event.html#返回值-2){.header-anchor} 返回值： {#返回值-2}

  类型     必填   说明
  -------- ------ -------------------------------
  Number   是     事件id，订阅失败返回undefined

#### [\#](event.html#示例-2){.header-anchor} 示例： {#示例-2}

::: {.language-javascript .extra-class}
``` language-javascript
const evtId = event.subscribe({
  eventName: 'myEventName',
  callback: function(res) {
    if (res.package === 'com.example.demo') {
      console.log(res.params)
    }
  }
})
console.log(evtId)
```
:::

### [\#](event.html#event-unsubscribe-object){.header-anchor} event.unsubscribe(OBJECT) {#event-unsubscribe-object}

取消订阅公共事件

#### [\#](event.html#参数-3){.header-anchor} 参数： {#参数-3}

  参数名   类型     必填   说明
  -------- -------- ------ --------
  id       Number   是     订阅id

#### [\#](event.html#示例-3){.header-anchor} 示例： {#示例-3}

::: {.language-javascript .extra-class}
``` language-javascript
const evtId = event.subscribe({
  eventName: 'myEventName',
  callback: function(res) {
    if (res.package === 'com.example.demo') {
      console.log(res.params)
    }
  }
})

event.unsubscribe({ id: evtId })
```
:::
