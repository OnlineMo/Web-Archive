# [\#](geolocation.html#地理位置-geolocation){.header-anchor} 地理位置 geolocation

## [\#](geolocation.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.geolocation" }
```
:::

## [\#](geolocation.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import geolocation from '@system.geolocation' 
// 或 
const geolocation = require('@system.geolocation')
```
:::

## [\#](geolocation.html#接口定义){.header-anchor} 接口定义

### [\#](geolocation.html#geolocation-getlocation-object){.header-anchor} geolocation.getLocation(OBJECT) {#geolocation-getlocation-object}

获取地理位置

#### [\#](geolocation.html#权限要求){.header-anchor} 权限要求

精确设备定位

开发者需要在 manifest.json 里面配置权限：

::: {.language-json .extra-class}
``` language-json
{
  "permissions": [
    { "name": "hapjs.permission.LOCATION" }
  ]
}
```
:::

#### [\#](geolocation.html#参数){.header-anchor} 参数：

  参数名     类型       必填   说明
  ---------- ---------- ------ -----------------------------------------
  timeout    Number     否     设置超时时间，单位是 ms，默认值为 30000
  success    Function   是     成功回调
  fail       Function   否     失败回调，可能是因为缺乏权限
  complete   Function   否     执行结束后的回调

#### [\#](geolocation.html#success-返回值){.header-anchor} success 返回值：

  参数名         类型                                       说明
  -------------- ------------------------------------------ --------------------------------------
  longitude      Number                                     经度，浮点数
  latitude       Number                                     纬度，浮点数
  altitude       Number                                     海拔、高度，单位m，浮点数
  speed          Number                                     速度值，单位m/s，浮点数
  accuracy       Number                                     精确度，值为正整数
  accuracyInfo   { horizontal: Number, vertical: Number }   精确度信息，包含水平和垂直方向精准度

#### [\#](geolocation.html#fail-返回错误代码){.header-anchor} fail 返回错误代码：

  错误码   说明
  -------- --------------
  203      该功能不支持
  204      超时返回

#### [\#](geolocation.html#示例){.header-anchor} 示例：

::: {.language-javascript .extra-class}
``` language-javascript
geolocation.getLocation({
  success: function(data) {
    console.log(
      `handling success: longitude = ${data.longitude}, latitude = ${
        data.latitude
      }, speed = ${data.speed}, altitude = ${data.altitude}`
    )
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}, errorMsg=${data}`)
  }
})
```
:::

### [\#](geolocation.html#geolocation-subscribe-object){.header-anchor} geolocation.subscribe(OBJECT) {#geolocation-subscribe-object}

监听地理位置。如果多次调用，仅最后一次调用生效

#### [\#](geolocation.html#权限要求-2){.header-anchor} 权限要求 {#权限要求-2}

精确设备定位

开发者需要在 manifest.json 里面配置权限：

::: {.language-json .extra-class}
``` language-json
{
  "permissions": [
    { "name": "hapjs.permission.LOCATION" }
  ]
}
```
:::

#### [\#](geolocation.html#参数-2){.header-anchor} 参数： {#参数-2}

  参数名     类型       必填   说明
  ---------- ---------- ------ ----------------------------------
  callback   Function   是     每次位置信息发生变化，都会被回调
  fail       Function   否     失败回调

#### [\#](geolocation.html#callback-返回值){.header-anchor} callback 返回值：

  参数名      类型     说明
  ----------- -------- ---------------------------
  longitude   Number   经度，浮点数
  latitude    Number   纬度，浮点数
  altitude    Number   海拔、高度，单位m，浮点数
  speed       Number   速度值，单位m/s，浮点数
  accuracy    Number   精确度，值为正整数

#### [\#](geolocation.html#fail-返回错误代码-2){.header-anchor} fail 返回错误代码： {#fail-返回错误代码-2}

  错误码   说明
  -------- --------------
  203      该功能不支持

#### [\#](geolocation.html#示例-2){.header-anchor} 示例： {#示例-2}

::: {.language-javascript .extra-class}
``` language-javascript
geolocation.subscribe({
  callback: function(data) {
    console.log(
      `handling success: longitude = ${data.longitude}, latitude = ${
        data.latitude
      }, speed = ${data.speed}, altitude = ${data.altitude}`
    )
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}, errorMsg=${data}`)
  }
})
```
:::

### [\#](geolocation.html#geolocation-unsubscribe){.header-anchor} geolocation.unsubscribe() {#geolocation-unsubscribe}

#### [\#](geolocation.html#权限要求-3){.header-anchor} 权限要求 {#权限要求-3}

精确设备定位

::: {.language-json .extra-class}
``` language-json
{
  "permissions": [
    { "name": "hapjs.permission.LOCATION" }

```
:::

#### [\#](geolocation.html#参数-3){.header-anchor} 参数： {#参数-3}

无

#### [\#](geolocation.html#示例-3){.header-anchor} 示例： {#示例-3}

