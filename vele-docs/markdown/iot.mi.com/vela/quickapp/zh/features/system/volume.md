# [\#](volume.html#系统音量-volume){.header-anchor} 系统音量 volume

## [\#](volume.html#接口声明){.header-anchor} 接口声明

::: {.language-json .extra-class}
``` language-json
{ "name": "system.volume" }
```
:::

## [\#](volume.html#导入模块){.header-anchor} 导入模块

::: {.language-javascript .extra-class}
``` language-javascript
import volume from '@system.volume' 
// 或 
const volume = require('@system.volume')
```
:::

## [\#](volume.html#接口定义){.header-anchor} 接口定义

### [\#](volume.html#volume-getmediavalue-object){.header-anchor} volume.getMediaValue (OBJECT) {#volume-getmediavalue-object}

获取当前多媒体音量

#### [\#](volume.html#参数){.header-anchor} 参数

  参数名     类型       必填   说明
  ---------- ---------- ------ ------------------
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](volume.html#success-返回值){.header-anchor} success 返回值：

  参数值   类型     说明
  -------- -------- --------------------------------
  value    Number   系统媒体当前音量，0.0-1.0 之间

#### [\#](volume.html#示例){.header-anchor} 示例

::: {.language-javascript .extra-class}
``` language-javascript
volume.getMediaValue({
  success: function(data) {
    console.log(`handling success: ${data.value}`)
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
:::

### [\#](volume.html#volume-setmediavalue-object){.header-anchor} volume.setMediaValue (OBJECT) {#volume-setmediavalue-object}

设置当前多媒体音量

#### [\#](volume.html#参数-2){.header-anchor} 参数 {#参数-2}

  参数名     类型       必填   说明
  ---------- ---------- ------ --------------------------
  value      Number     是     设置的音量，0.0-1.0 之间
  success    Function   否     成功回调
  fail       Function   否     失败回调
  complete   Function   否     执行结束后的回调

#### [\#](volume.html#fail-返回值){.header-anchor} fail 返回值：

[支持通用错误码](../grammar.html#通用错误码)

#### [\#](volume.html#示例-2){.header-anchor} 示例 {#示例-2}

::: {.language-javascript .extra-class}
``` language-javascript
volume.setMediaValue({
  value: 0.5,
  success: function() {
    console.log('handling success')
  },
  fail: function(data, code) {
    console.log(`handling fail, code = ${code}`)
  }
})
```
:::

## [\#](volume.html#事件){.header-anchor} 事件

### [\#](volume.html#volume-onmediavaluechanged){.header-anchor} volume.onMediaValueChanged {#volume-onmediavaluechanged}

多媒体音量发生变化事触发

#### [\#](volume.html#回调object参数){.header-anchor} 回调Object参数：

  参数名   类型     必填   说明
  -------- -------- ------ --------------------------------------
  value    Number   是     系统媒体当前音量，范围：0.0-1.0 之间

#### [\#](volume.html#示例-3){.header-anchor} 示例 {#示例-3}

::: {.language-javascript .extra-class}
``` language-javascript
volume.onMediaValueChanged = function(res) {
  console.log('volume media value changed:', res.value)
}
```
