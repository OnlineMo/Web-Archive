# [\#](toolkit-params.html#编译参数){.header-anchor} 编译参数

编译工具提供了多种编译能力，开发者可以根据项目需求进行设置。请**注意**，编译参数仅在
`build`、`server`、`release` 命令中可用

## [\#](toolkit-params.html#如何设置编译参数){.header-anchor} 如何设置编译参数

通常有两种方式设置编译参数，以抽取单独的 source-map 文件为例：

-   在命令行携带编译参数

::: {.language-shell .extra-class}
``` language-shell
aiot build --devtool=source-map
```
:::

-   在项目根目录新建配置文件 quickapp.config.js，并配置 cli 属性；

::: {.language-js .extra-class}
``` language-js
module.exports = {
  cli: {
    devtool: "source-map",
  },
};
```
:::

## [\#](toolkit-params.html#查看当前工具支持的全部编译参数){.header-anchor} 查看当前工具支持的全部编译参数

::: {.language-shell .extra-class}
``` language-shell
 npx aiot build -h
```
:::
